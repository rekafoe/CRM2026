## Как использовать

### Создание продукта через операции

1. **Создать операции** (если еще нет):
```bash
POST /api/operations
{
  "name": "Цифровая печать A4",
  "operation_type": "print",
  "price": 0.10,
  "price_unit": "per_sheet",
  "setup_cost": 0,
  "parameters": {"color_mode": "cmyk"}
}
```

2. **Привязать операции к продукту**:
```bash
POST /api/products/1/operations
{
  "operation_id": 1,
  "sequence": 1,
  "is_required": true,
  "price_multiplier": 1.0
}
```

3. **Рассчитать цену** (через `FlexiblePricingService`):
```typescript
const result = await FlexiblePricingService.calculatePrice(
  productId,
  configuration,
  quantity
);
```

## Пример расчета

### Визитки 90x50, 100 шт

**Операции:**
- Печать цифровая: 10 листов × 0.15 = 1.50 BYN
- Резка: 10 листов × 0.01 + 5 (приладка) = 5.10 BYN
- Материал: 10 листов × 0.40 = 4.00 BYN

**Итого:**
- Себестоимость: 10.60 BYN
- Наценка 2.2×: 23.32 BYN
- Цена за визитку: 0.23 BYN
