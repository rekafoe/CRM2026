# 📊 Reports - Отчеты и аналитика

Система отчетности и аналитики для мониторинга работы CRM.

## 🎯 Основные функции

- Ежедневные отчеты по заказам
- Аналитика по пользователям
- Отчеты по материалам
- PDF генерация отчетов
- Экспорт данных

## 🏗️ Архитектура

### Backend
- **Контроллеры:**
  - `backend/src/controllers/materialReportController.ts`
  - `backend/src/controllers/supplierAnalyticsController.ts`
  - `backend/src/controllers/materialsAnalyticsController.ts`
- **Сервисы:**
  - `backend/src/services/materialReportService.ts`
  - `backend/src/services/pdfReportService.ts`
  - `backend/src/services/supplierAnalyticsService.ts`
- **Роуты:** `backend/src/routes/reports.ts`

### Frontend
- **Страницы:**
  - `frontend/src/pages/AdminReportsPage.tsx`
  - `frontend/src/pages/DailyReportPage.tsx`
  - `frontend/src/pages/ReportDetailPage.tsx`

## 📊 Типы отчетов

### Ежедневные отчеты
- Заказы за день
- Выручка
- Статистика по пользователям
- Потребление материалов

### Материалы
- Остатки на складе
- Движения материалов
- Низкие остатки
- История поставок

### Поставщики
- Аналитика поставок
- Сравнение цен
- Надежность поставщиков
- Рекомендации

## 📄 PDF отчеты

### Генерация
- **Технология:** Puppeteer
- **Сервис:** `backend/src/services/pdfReportService.ts`
- **Формат:** HTML → PDF

### Типы PDF
- Отчет по остаткам
- Ежедневный отчет
- Аналитика по материалам

## 🎨 Интерфейс

### Админ панель
- **📊 Ежедневные отчёты** - Кнопка в топбаре (только для админов)
- Страница с отчетами
- Фильтры по датам и пользователям

### Telegram бот
- `/stock_pdf` - PDF отчет по остаткам
- Автоматическая отправка отчетов

## 🔧 API Endpoints

### Отчеты
- `GET /api/reports/daily` - Ежедневный отчет
- `GET /api/reports/materials` - Отчет по материалам
- `GET /api/reports/suppliers` - Отчет по поставщикам

### PDF
- `GET /api/reports/stock-pdf` - PDF отчет по остаткам
- `POST /api/reports/generate-pdf` - Генерация PDF

### Аналитика
- `GET /api/analytics/materials` - Аналитика материалов
- `GET /api/analytics/suppliers` - Аналитика поставщиков

## 📈 Метрики

### Заказы
- Количество заказов
- Выручка
- Средний чек
- Конверсия

### Материалы
- Остатки
- Расходы
- Тренды потребления
- Прогнозы

### Пользователи
- Активность
- Производительность
- Статистика по ролям

### Зарплаты сотрудников
- Фонд ЗП к выплате с учетом премий и штрафов
- Динамика начислений по месяцам
- Рейтинг сотрудников по выплатам и ставке в час
- Разрез фонда ЗП по ролям
- Контрольные сигналы: начисления без часов, активные без начислений, высокая доля штрафов
